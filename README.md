# AdventureWorks Excel Sales Dashboard

## Project Overview
This project demonstrates an end-to-end data analytics workflow using Microsoft Excel. Raw AdventureWorks sales data was cleaned, transformed, and analyzed to build an interactive dashboard for business decision-making.

## Objective
To convert raw transactional data into a clear and interactive Excel dashboard that helps analyze:
- Sales performance
- Product trends
- Customer behavior
- Regional insights

## Data Source
- AdventureWorks dataset (raw files provided in ZIP format)
- Multiple related tables including sales, products, customers, and geography

## Tools & Skills Used
- Microsoft Excel
- Power Query (data cleaning & transformation)
- Pivot Tables & Pivot Charts
- Calculated fields and Excel formulas
- Dashboard design and KPI visualization

## Key Steps Performed
- Extracted raw data from ZIP archive
- Cleaned and structured data using Power Query
- Created relationships across multiple datasets
- Built KPIs such as total sales, order volume, and category-wise performance
- Designed an interactive Excel dashboard with slicers and filters

## Final Output
An interactive Excel dashboard that enables quick analysis of sales performance by:
- Product categories
- Customers
- Regions
- Time periods
